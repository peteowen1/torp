# Check if current throw-in should be mirrored based on team change

Check if current throw-in should be mirrored based on team change

## Usage

``` r
is_current_throw_in_team_change(throw_in, team_id_mdl)
```

## Arguments

- throw_in:

  A vector indicating if the play is a throw-in.

- team_id_mdl:

  A vector of team IDs for modeling.

## Value

Logical vector indicating mirror condition
