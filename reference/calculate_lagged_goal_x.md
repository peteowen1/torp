# Calculate Lagged Goal X

Calculate Lagged Goal X

## Usage

``` r
calculate_lagged_goal_x(goal_x, team_id_mdl, venue_length, lag = 1)
```

## Arguments

- goal_x:

  A vector of goal x-coordinates.

- team_id_mdl:

  A vector of team IDs for modeling.

- venue_length:

  The length of the venue.

- lag:

  The number of positions to lag (default is 1).

## Value

A vector of lagged goal x-coordinates.
