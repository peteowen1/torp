# Maximum reasonable Euclidean distance between consecutive PBP coordinates (metres). Jumps exceeding this in pitch-relative space are smoothed via neighbor interpolation. An AFL field is ~165m long and ~135m wide; 100m covers most realistic play distances.

Maximum reasonable Euclidean distance between consecutive PBP
coordinates (metres). Jumps exceeding this in pitch-relative space are
smoothed via neighbor interpolation. An AFL field is ~165m long and
~135m wide; 100m covers most realistic play distances.

## Usage

``` r
COORD_JUMP_THRESHOLD
```

## Format

An object of class `numeric` of length 1.
