# Rating shift fraction per round After each simulated game, team ratings are shifted by this fraction of (actual_result - expected_result). Controls how much in-season results update the "hot" rating used for subsequent rounds.

Rating shift fraction per round After each simulated game, team ratings
are shifted by this fraction of (actual_result - expected_result).
Controls how much in-season results update the "hot" rating used for
subsequent rounds.

## Usage

``` r
SIM_RATING_SHIFT
```

## Format

An object of class `numeric` of length 1.
