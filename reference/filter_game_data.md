# Filter game data

Filter game data

## Usage

``` r
filter_game_data(df, season_val, round_num, matchid, team)
```

## Arguments

- df:

  Input data frame

- season_val:

  Season value

- round_num:

  Round number

- matchid:

  Match ID (NULL for no filtering)

- team:

  Team name (NULL for no filtering)

## Value

Filtered data frame
