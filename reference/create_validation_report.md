# Create Model Validation Report

This function is intended for internal use and may be unexported in a
future release. Generates a comprehensive model validation report

## Usage

``` r
create_validation_report(evaluation_results, comparison_results = NULL)
```

## Arguments

- evaluation_results:

  List of evaluation results

- comparison_results:

  Dataframe of model comparisons (optional)

## Value

Character string containing formatted report
