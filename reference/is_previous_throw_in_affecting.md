# Check if previous throw-in affects current play mirroring

Check if previous throw-in affects current play mirroring

## Usage

``` r
is_previous_throw_in_affecting(throw_in, team_id_mdl, x)
```

## Arguments

- throw_in:

  A vector indicating if the play is a throw-in.

- team_id_mdl:

  A vector of team IDs for modeling.

- x:

  A vector of x-coordinates.

## Value

Logical vector indicating mirror condition
