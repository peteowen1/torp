# Calculate time remaining as a proportion of total game time

Calculate time remaining as a proportion of total game time

## Usage

``` r
calculate_time_remaining_pct(period, period_seconds)
```

## Arguments

- period:

  Numeric vector of period numbers (1-4)

- period_seconds:

  Numeric vector of seconds elapsed in the current period

## Value

Numeric vector of proportions (0 to 1)
