# Default total predicted TOG for league-wide centering (18 players per team x 18 teams). Used as fallback; actual centering adapts to the number of teams present.

Default total predicted TOG for league-wide centering (18 players per
team x 18 teams). Used as fallback; actual centering adapts to the
number of teams present.

## Usage

``` r
TOTAL_PRED_TOG
```

## Format

An object of class `integer` of length 1.
