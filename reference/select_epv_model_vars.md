# Select EPV Model Variables

This function is intended for internal use and may be unexported in a
future release.

## Usage

``` r
select_epv_model_vars(df, label = FALSE)
```

## Arguments

- df:

  A dataframe containing cleaned play-by-play data.

- label:

  Logical, whether to include the label variable for training.

## Value

A dataframe with selected variables for EPV modeling.
