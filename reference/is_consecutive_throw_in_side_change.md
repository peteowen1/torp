# Check if consecutive throw-ins should be mirrored based on position change

Check if consecutive throw-ins should be mirrored based on position
change

## Usage

``` r
is_consecutive_throw_in_side_change(throw_in, team_id_mdl, x)
```

## Arguments

- throw_in:

  A vector indicating if the play is a throw-in.

- team_id_mdl:

  A vector of team IDs for modeling.

- x:

  A vector of x-coordinates.

## Value

Logical vector indicating mirror condition
