# Default EPV assignment parameters

Returns a named list of all EPV assignment parameters with their default
values. Used by
[`create_player_game_data()`](https://peteowen1.github.io/torp/reference/create_player_game_data.md)
when no custom params are provided.

## Usage

``` r
default_epv_params()

default_credit_params()
```

## Value

A named list of EPV assignment parameters.
