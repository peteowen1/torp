# Get Round Games

This function is intended for internal use and may be unexported in a
future release. Retrieves game data for a specific round in a season.

## Usage

``` r
get_round_games(season, round)
```

## Arguments

- season:

  The AFL season year (numeric).

- round:

  The round number (numeric).

## Value

A dataframe containing game data for the specified round.
