# Chains descriptions for the contest target / kick result side These appear in the row BEFORE the contest outcome (Spoil, Mark, etc.) at the same x,y coordinates.

same x,y coordinates.

## Usage

``` r
CHAINS_CONTEST_TARGET_DESCS
```

## Format

An object of class `character` of length 2.
