# Check if throw-in two plays ago affects current play

Check if throw-in two plays ago affects current play

## Usage

``` r
is_throw_in_two_ago_affecting(throw_in, team_id_mdl, x)
```

## Arguments

- throw_in:

  A vector indicating if the play is a throw-in.

- team_id_mdl:

  A vector of team IDs for modeling.

- x:

  A vector of x-coordinates.

## Value

Logical vector indicating mirror condition
