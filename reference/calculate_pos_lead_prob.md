# Calculate Positive Lead Probability

Calculate Positive Lead Probability

## Usage

``` r
calculate_pos_lead_prob(
  points_diff,
  opp_goal,
  opp_behind,
  no_score,
  behind,
  goal
)
```

## Arguments

- points_diff:

  The points difference.

- opp_goal:

  Opponent's goal probability.

- opp_behind:

  Opponent's behind probability.

- no_score:

  No score probability.

- behind:

  Behind probability.

- goal:

  Goal probability.

## Value

A numeric value representing the positive lead probability.
